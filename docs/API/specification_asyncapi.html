API для взаимодействия с заявками

Созздание заявки

Request Body schema: application/json
required
id
string

Идентификатор заявки

organizationId
string

Идентификатор организации

taskDescription
string

Описание функционала волонтеров

maxVolunteers
integer

Макисмальное количество волонтеров на выполнение заявки

timeSlotbeg
string

Время начала волонтерства по заявке

timeSlotend
string

Время окнончания волонтерства по заявке

Responses

rabbitmq://localhost/volunteerApplications
